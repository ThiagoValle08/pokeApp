Proyecto en Desarrollo

Este proyecto es una aplicación que consume la API de Pokémon.

Características:

  • Desarrollado en Angular.
  • Interfaz construida con CSS, Angular Material y PrimeNG.
  • Datos almacenados en LocalStorage.
  • Actualmente en fase de demostración.

Este es un desarrollo en curso, aún se está optimizando su funcionamiento.
